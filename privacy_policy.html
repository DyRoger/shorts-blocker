Privacy Policy

Last updated: 2025-08-21

This privacy policy describes how the YouTube Shorts Blocker extension ("the Extension") handles data.

Data collection

The Extension runs locally in your browser and does not transmit any browsing data to external servers. It uses the following browser APIs:

  • storage — to save local settings and the counter that tracks how many unique Shorts videos were blocked.
  • tabs — to detect and refresh YouTube tabs when the user toggles the blocker.
  • Content script access to https://www.youtube.com/* pages to detect and hide Shorts UI elements and to intercept Shorts URLs for local redirecting.

How data is used

All processing happens locally in the browser. The counter stored via the storage API contains only YouTube video IDs (used to deduplicate counts) and no user-identifying data. No analytics, crash reports, or remote logging is performed.

Remote code

The Extension does not download or execute remote code at runtime. All code is packaged inside the extension bundle.

Third-party services

None.

User control

You can disable or remove the Extension at any time. You may clear the stored counter and settings from the browser's extension storage or by removing the extension.

Contact

For questions, open an issue on the project's GitHub: https://github.com/DyRoger/shorts-blocker
